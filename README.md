# MétéoPop Festival :

Bienvenue sur le site officiel du festival **MétéoPop** ! Ce site permet de consulter la programmation, les informations pratiques, les tarifs de la billetterie, et de contacter les organisateurs.


## Structure du projet : 

meteo-site/
│── index.html
│── contact.html
│── styles.css
│── styles.css
│── assets/
│── README.md

## Installation : 

1. **Cloner le dépôt** :
   ```bash
   git clone https://github.com/ton-utilisateur/meteopop.git

Ouvrir le projet :

Ouvre le dossier meteo-site dans ton éditeur de code préféré (VS Code, Sublime Text, etc.).

Lancer le site :

Utilise un serveur local pour visualiser le site. Par exemple, avec l'extension Live Server dans VS Code :

Fais un clic droit sur index.html.
Sélectionne "Open with Live Server".
